## 二、圆角规范

全局**大圆角设计**，模拟实体扭蛋机的柔和触感，无尖锐直角。

| 等级 | 值 | 用途 |
|---|---|---|
| 卡片圆角 | `12px` (`rounded-[12px]`) | 灵感卡片本体 |
| 标准圆角 | `16px` (`rounded-2xl`) | 按钮、卡片文字遮罩、输入框 |
| 大圆角 | `24px` (`rounded-3xl`) | 弹窗底部、抽屉顶部 |
| 全圆角 | `9999px` (`rounded-full`) | 图标按钮、小圆点、球体 |
| 机器穹顶 | `rounded-t-[100px]` | 扭蛋机玻璃罩顶部 |
| 机器底座 | `rounded-b-3xl` | 扭蛋机底座底部 |

## 五、弹窗与抽屉规范

### 5.1 扭蛋结果弹窗
- 覆盖全屏，居中弹出
- 背景：`bg-black/20 backdrop-blur-sm`
- 动画：3D旋转翻牌（`rotateY: 0→90→0`）
- 卡片尺寸：`max-w-[331px]`
- 关闭按钮：右上角浮层圆形按钮

### 5.2 标记弹窗
- 底部弹出式（移动端）/ 居中弹出（桌面端）
- 背景：`bg-black/40 backdrop-blur-sm`
- 圆角：`rounded-t-3xl sm:rounded-3xl`
- 宽度：`max-w-md`
- 弹入动画：`y: 100% → 0`

### 5.3 今日手气抽屉
- 固定底部，最大高度 `max-h-[80vh]`
- 背景：`bg-white rounded-t-3xl`
- 内容：双列网格，点击跳转卡片详情
- 遮罩：`bg-black/20 backdrop-blur-sm`

## 六、扭蛋机视觉规范

### 6.1 机器结构
- 玻璃罩：`w-64 h-80`，`bg-white/60 backdrop-blur-xl`
- 玻璃反射效果：顶部左侧半透明白色模糊块
- 底座：`h-24 bg-[#E3E8C8] rounded-xl`，内含旋钮
- 球体：颜色取自4种渐变色（见1.1色板）

### 6.2 按钮规范
- 主按钮：「扭一下 (N/10)」圆角全满，白色背景，暖棕文字
- 禁用态：`bg-gray-200 text-gray-400`，`cursor-not-allowed`
- 满限态：文字变为「今日灵感够多啦，出门走走吧」
- 今日手气入口：底部居中 `text-sm text-[#8D9876]/60`
- 所有按钮点击：`active:scale-95` 缩放反馈

## 七、卡片视觉规范

### 7.1 卡片物理尺寸
- 宽高比：**331:460**（`aspect-[331/460]`）
- 3D透视：`perspective-[1500px]`
- 翻转动画时长：**600ms**，`easeInOut`

### 7.2 普通款卡片设计
- 正面：Figma `111Vectorized` 手绘肌理图案
- 背面：Figma `121Vectorized` 手绘肌理图案
- 标题在下部居中，`font-serif`，`text-[#5C5446]`

### 7.3 稀有款卡片设计
- 正面：Figma `211Vectorized` 手绘肌理图案
- 背面：Figma `221Vectorized` 手绘肌理图案
- 左上角徽章：「稀有款」白底绿字圆角条

### 7.4 限定款卡片设计
- 正面：Figma `311Vectorized` 手绘肌理图案
- 背面：Figma `321Vectorized` 手绘肌理图案
- 左上角徽章：「限定款」白底蓝灰字圆角条

### 7.5 小尺寸卡片（缩略图）
- 卡片文字区域叠加半透明白色毛玻璃层
- 标题：`text-xs`，`font-serif`，双行截断
- 描述：`text-[9px]`，三行截断

### 7.6 翻转动画序列
```
抽卡触发 → 展示正面（稍带旋转扭动效果，400ms）
        → 悬停800ms
        → 自动翻转到背面（600ms）
        → 触发「去过啦」按钮显示
```

## 八、动效设计原则

1. **速率偏缓、曲线柔和**：无快速闪烁、剧烈震动刺激效果
2. **核心动效**集中在扭蛋掉落、卡片弹出两个环节
3. **次级交互**：页面切换、按钮点击采用轻微回弹动效（`spring`）
4. **过渡时长参考**：
   - 扭蛋机摇晃：`0.5s`，重复2次
   - 球体下落弹入：`spring` (`bounce: 0.4`)，延迟`0.6s`
   - 卡片弹出：`spring` (`stiffness: 150, damping: 20`)
   - 弹窗划入：`spring` (`damping: 25, stiffness: 200`)
   - 页面过渡：`framer-motion` 渐变动画
   - Toast显示：3秒自动消失

## 九、响应式规范

- 移动端优先设计
- 桌面端弹窗居中显示（`sm:items-center sm:rounded-3xl`）
- 卡片详情页：`max-w-[331px]` 居中
- 收藏页/卡片墙：`grid-cols-2` 双列网格
- 内容区域：`min-h-screen` + `pt-20`（避开固定导航）
