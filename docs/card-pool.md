# Outspire 卡片池文档（Card Pool）

- 版本：V1.0
- 卡片总数：当前 13 张 → 目标 50+ 张
- 稀有度分布：普通 ~70% / 稀有 ~25% / 限定 ~5%
- 审核状态：待扩充审查

---

## 一、卡片数据结构

```typescript
export type Rarity = 'normal' | 'rare' | 'limited';
export type Condition = 'rain' | 'night' | 'sunset' | 'spring' | 'summer' | 'autumn' | 'winter' | 'fog' | 'snow';

export interface InspirationCard {
  id: string;        // 唯一标识，如 'n1', 'r1', 'l1'
  title: string;     // 卡片标题
  description: string; // 探索灵感描述（建议 20-50 字）
  rarity: Rarity;    // 稀有度
  condition?: Condition; // 限定卡触发条件
  icon: string;      // lucide-react 图标名
}
```

---

## 二、稀有度分级说明

| 等级 | 占比 | 定位 | 规则 |
|---|---|---|---|
| 普通款 (Normal) | ~70% | 日常通用型探索 | 不受环境限制，随时可抽到 |
| 稀有款 (Rare) | ~25% | 更小众、更有体验感的探索 | 不受环境限制，概率较低 |
| 限定款 (Limited) | ~5% | 仅在特定环境条件下入池 | 需匹配 `condition` 才进入候选池 |

## 六、限定卡条件匹配规则（强绑定）

**限定卡必须严格匹配用户所在地的实时环境条件**，环境条件从免费天气 API（Open-Meteo）获取，匹配通过后才进入候选池。不匹配时该限定卡不会出现在任何抽卡结果中。

```typescript
// 环境数据来源：Open-Meteo API（每日调用，无需 API Key）
// - 天气：根据天气代码判断（晴天、下雨、起雾、下雪等）
// - 时段：根据当前小时数判断
// - 季节：根据当前月份判断

export interface Environment {
  weather: 'clear' | 'cloudy' | 'rain' | 'drizzle' | 'thunderstorm' | 'fog' | 'haze' | 'mist' | 'snow' | 'drifting-snow';
  timeOfDay: 'day' | 'night' | 'sunset';
  season: 'spring' | 'summer' | 'autumn' | 'winter';
}

const conditionMatch = (card: InspirationCard, env: Environment): boolean => {
  switch (card.condition) {
    case 'rain':   return ['rain', 'drizzle', 'thunderstorm'].includes(env.weather);
    case 'fog':    return ['fog', 'haze', 'mist'].includes(env.weather);
    case 'snow':   return ['snow', 'drifting-snow'].includes(env.weather);
    case 'night':  return env.timeOfDay === 'night';                  // 晚8点至早5点
    case 'sunset': return env.timeOfDay === 'sunset';                 // 约日落前后1小时
    case 'spring': return env.season === 'spring';                    // 3-5月
    case 'summer': return env.season === 'summer';                    // 6-8月
    case 'autumn': return env.season === 'autumn';                    // 9-11月
    case 'winter': return env.season === 'winter';                    // 12-2月
    default:       return false;
  }
};
```

**关键原则：**
1. 限定卡**不降级** — 如果当前环境没有匹配的限定卡，则限定卡池为空，该次抽卡仅在普通+稀有中抽取（不会把不匹配的限定卡降级为稀有/普通放出）
2. 限定卡不会在没有对应条件的任何时候被抽到
3. 限定卡的数量设计上保证每个常见条件至少匹配1张（如雨天、夜间、日落、四季），确保限定卡池大多数时间不为空
